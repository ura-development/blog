Dumbshits guide to flashing a keyboard

Starting notes
- hex file: a file that ends with '.hex'.
- reset button: the nub thingy at the back of your kb, if you can't see it, see anywhere with 2 holes saying 'rst' or 'reset'
- some keyboards are already flashed with a via hex and you just have to download via and sometimes upload a '.json' file


Step 1:

Download the keyboards hex file you need. If you are wanting via for your board download off.
https://caniusevia.com/docs/download_firmware#0 if its on the list or if its said to be via compatible the product page/owners resources (not all boards have via compatability).


Step 2:

Download qmk toolbox for your os (supports unix, windows and mac) and install it. Make sure to say yes to installing drivers.


Step 3:

Open up qmk toolbox and select your boards microcontroller.
Most boards are atmega32a4's.


Step 4:

Find the keyboards "reset" button on the pcb or sites mentioned keys to hold for resetting. For holding keys you have to hold the key or combination while plugging in the keyboard.
For the reset button just press it.
After the keyboard has a reset it should show this


Step 5:

Select the file on your computer that has been downloaded from step 1



If you have done all the steps before press flash.
It should show up like this


You now have a flashed keyboard.
unplug and replug and it should be working as intended.
